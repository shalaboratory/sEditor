sEditor.js v3.0.0

A lightweight, zero-dependency WYSIWYG rich text editor for modern web applications.

Features

  • Formatting: Bold, Italic, Underline, Strikethrough, Super/Subscript.
  • Layout: Text alignment (Left, Center, Right, Justify) and Lists.
  • Multimedia: Advanced Image and Link insertion modals.
  • Theme: Built-in Dark Mode and Fullscreen mode.
  • Development: Direct HTML source code editing.

Quick Start

1. Link the script in your HTML:

<script src="seditor_v3.min.js"></script>

2. Initialize the editor:

const editor = new sEditor();
editor.run({
  // REQUIRED: The CSS selector for the container element.
  selector: '#myEditor',

  // Enable or disable features. All are true by default.
  plugins: {
    image: true,   // Image insertion modal
    link: true,    // Link insertion modal
  },

  // Initialize the editor in dark mode. Default: false.
  darkMode: false,


});

API Reference

Method Description
getContent() Returns the current content as an HTML string.
setContent(html) Replaces editor content with the provided HTML.
toggleDarkMode() Toggles the interface between light and dark themes.
toggleFullScreen() Toggles the editor between normal and full-page mode.

Shortcuts

  • Ctrl + B: Bold
  • Ctrl + I: Italic
  • Ctrl + U: Underline
  • Ctrl + Z / Y: Undo / Redo
